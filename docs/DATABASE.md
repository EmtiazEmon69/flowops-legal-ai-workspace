# Database Notes

Core tables for the MVP:

- `clients`: client and account profile data
- `documents`: uploaded files, document categories, extracted text, summaries
- `reminders`: due dates, channels, status

Future tables:

- `users`
- `teams`
- `roles`
- `matters`
- `workflow_runs`
- `messages`
- `audit_events`
- `document_embeddings`
