# Roadmap

## Phase 1: MVP Foundation

- Build dashboard UI
- Create client and document APIs
- Add AI drafting
- Add reminder tracking
- Store data in PostgreSQL or SQLite for local development

## Phase 2: Document Intelligence

- Add OCR processing
- Extract structured fields
- Generate document summaries
- Add confidence scores and review states

## Phase 3: Automations

- Email reminders
- SMS and WhatsApp integration
- Missing document follow-ups
- Deadline alerts

## Phase 4: Search and Analytics

- Add embeddings
- Connect vector search
- Show time saved, overdue work, and staff productivity
