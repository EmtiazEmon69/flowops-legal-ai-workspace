# FlowOps AI MVP

## First Build

The first sellable version should help document-heavy teams move faster without removing human review.

### Included

- Client profiles and matter tracking
- Document upload queue
- OCR and AI extraction pipeline placeholder
- AI drafting endpoint
- Reminder creation and status tracking
- Dashboard metrics
- Natural language search endpoint placeholder

### Not Included Yet

- Billing
- Full role-based permissions
- Production-grade document storage
- SMS and WhatsApp providers
- Vector database
- Workflow builder UI

## User Workflow

1. Staff creates a client.
2. Staff uploads documents.
3. AI extracts text and key fields.
4. System creates missing-document and due-date reminders.
5. Staff asks AI to generate a summary, email, or draft.
6. Human reviews and approves before sending.
